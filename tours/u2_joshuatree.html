Voltar para Shows e Turnês

U2 – "The Joshua Tree Tour"

2017-2019

Visão Geral

A turnê comemorativa do U2 para o álbum The Joshua Tree apresentou uma tela LED 4K de 61 m x 14 m (200 x 45 pés) – uma das maiores já utilizadas em turnê até aquele momento.

Cliente: U2
Solução Disguise: Servidores gx 2
Designer do Show: Willie Williams

O Desafio

Criar uma experiência visual imersiva para celebrar um álbum icônico do U2, utilizando uma das maiores telas LED já usadas em turnês até então. O desafio incluía a necessidade de coordenar um enorme volume de conteúdo visual 4K em um painel panorâmico massivo, garantindo sincronização perfeita e confiabilidade em shows ao redor do mundo.

Todo o conteúdo para esta tela panorâmica foi gerenciado pelos servidores de mídia Disguise gx 2. O designer de longa data da banda, Willie Williams, contou com a pré-visualização 3D do Disguise para mapear as impressionantes imagens do deserto em toda a tela gigante.

A Solução Disguise

A produção utilizou servidores Disguise gx 2 para alimentar o que era, na época, a maior produção de turnê baseada em LED do mundo. A plataforma Disguise permitiu:

  • Sincronização perfeita de conteúdo 4K em uma tela gigante de 61 m x 14 m
  • Visualização 3D antecipada do design completo do palco para planejamento preciso
  • Integração com o sistema de iluminação para criar uma experiência visual coesa
  • Redundância robusta para garantir performances sem falhas em dezenas de shows
  • Adaptabilidade para ajustar conteúdo visual para diferentes locais e arenas
